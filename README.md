Spectrumyzer
==============

Dependencies
--------------

Make sure you have `git` and the full `gcc` stack installed, and then install the following packages: 

```
python2
fftw
libpulse
```

Build
--------------

    make clean all

If this completes successfully, test with:

    ./impulse.py

And try to play some music.
